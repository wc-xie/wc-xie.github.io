降维的几种方法

1. 数据降维概论

1. Intoduction

1.1. 降维的目的

很多实验所需要的数据均为高维数据，也就是多列变量值决定目标值。

高维数据信息量大。高维数据不仅仅包含了各个维度的表观信息，还有数据内部的隐含信息，包括维度之间的耦合。

高维度数据的缺点也很明显，消耗计算资源，计算时间大，同时使得冗余且耦合的数据对实验结果造成影响，甚至造成“维度灾难”。

更重要的是，高维数据对数据的integration 和 intepretion造成了挑战。

1.2. 降维的实质

降维度的实质，包括以下内容：

  • 减少特征属性的个数
  • 剔除不相关或冗余特征
  • 减少特征个数
  • 减少runtime
  • 确保特征属性之间是相互独立的

2. 特征选择

数据降维分为特征选择和特征提取两类

2.1. 低方差滤波

低方差滤波的原理很简单：直接删除数据方差少的那一列数据。

该方法假设数据列变化非常小的列包含的信息量少。因此，所有的数据列方差小的列被移除。

2.1.1. 实施步骤

  1. 估算/检查缺失值是否已经被填充
  2. 再计算所有数值变量的方差
  3. 删除方差非常小的值即可

2.2. 高相关滤波

与低方差滤波相反，高相关滤波认为当两列数据变化趋势相似时，它们包含的信息也相似。因此，当两列相似时，可删除其中一列。

对于数值列之间的相似性通过计算相关系数来表示，对于名词类列的相关系数可以通过计算皮尔逊卡方值来表示。相关系数大于某个阈值的两列只保留一列。

计算方差时，应注意方差与数据范围相关的，因此在采用该方法前需要对数据做归一化处理

2.2.1. 实施步骤

  1. 首先，删除因变量，并将剩余的变量保存在新的数据列中
  2. 计算两两数据之间关联性
  3. 如果一对变量之间的相关性大于0.5-0.6，则为高相关变量
  4. 选择其中一种变量删除

2.3. 随机森林 （组合决策树）

一种常用的降维方法是对目标属性产生许多巨大的树，然后根据对每个属性的统计结果找到信息量最大的特征子集。

例如，我们能够对一个非常巨大的数据集生成非常层次非常浅的树，每颗树只训练一小部分属性。如果一个属性经常成为最佳分裂属性，那么它很有可能是需要保留的信息特征。对随机森林数据属性的统计评分会向我们揭示与其它属性相比，哪个属性才是预测能力最好的属性。

2.3.1. 实施步骤

  1. 在开始降维前，我们先把数据转换成数字格式，因为随机森林只接受数字输入
  2. 我们可以把数据序号这列删除，这个不影响结果
  3. 拟合模型后，根据特征的重要性绘制成图
  4. 手动选择最顶层的特征来减少数据集中的维度

3. 特征提取

3.1. 反向特征消除

该方法的实质是，从模型结果判断维度贡献。

先通过所有数据集建立模型。第一轮迭代中，依次删除一个维度，进行训练。将删除维度后的数据集的训练结果与全数据集的训练结果进行比较，选出删除后对结果影响较小的那个维度，作为降维后的数据集。第二轮迭代则删除2个维度，以此类推。

3.1.1. 实施步骤

  1. 先获取数据集中的全部n个变量，然后用它们训练一个模型
  2. 计算模型的性能
  3. 在删除每个变量（n次）后计算模型的性能，即我们每次都去掉一个变量，用剩余的n-1个变量训练模型
  4. 确定对模型性能影响最小的变量，把它删除
    重复此过程，直到不再能删除任何变量

3.2. 前向特征选择

前向特征构建是反向特征消除的反过程。在前向特征过程中，我们从 1 个特征开始，每次训练添加一个让分类器性能提升最大的特征。

前向特征构造和反向特征消除都十分耗时。它们通常用于输入维数已经相对较低的数据集。

3.2.1. 实施步骤

  1. 选择一个特征，用每个特征训练模型n次，得到n个模型
  2. 选择模型性能最佳的变量作为初始变量
  3. 每次添加一个变量继续训练，重复上一过程，最后保留性能提升最大的变量
  4. 一直添加，一直筛选，直到模型性能不再有明显提高

3.3. 缺失值比率

该方法基于：太多缺失值的数据列包含有用信息的可能性较少。在一个数据集中，若存在较多缺失值（可设定阈值以量化），可认为该数据集有用信息较少。

经验性方法。

3.3.1. 实施步骤

  1. 检查每个变量中缺失值的占比
  2. 缺失值很少。我们设阈值为20%
  3. 删除高于阈值的列

3.4. 经典：主成分分析（PCA）

PCA的主要思想是：将n维特征映射到k维上。这k维是全新的正交特征也被称为主成分，是在原有n维特征的基础上重新构造出来的k维特征。

3.4.1. PCA的工作原理

  1. 从原始的空间中顺序地找一组相互正交的坐标轴
  2. 第一个新坐标轴选择是原始数据中方差最大的方向
  3. 第二个新坐标轴选取是与第一个坐标轴正交的平面中使得方差最大的，第三个轴是与第1,2个轴正交的平面中方差最大的。
  4. 依次类推，可以得到n个这样的坐标轴。忽略方差几乎为0的坐标轴。保留方差大的坐标轴。

这相当于只保留包含绝大部分方差的维度特征，而忽略包含方差几乎为0的特征维度，实现对数据特征的降维处理。

3.4.2. 实施步骤

  1. 去除平均值
  2. 计算协方差矩阵
    3 计算协方差矩阵的特征值和特征向量
  3. 将特征值排序
  4. 保留前N个最大的特征值对应的特征向量
  5. 将原始特征转换到上面得到的N个特征向量构建的新空间中（最后两步，实现了特征压缩）

3.4.3. 特点

只关⼼使变量的⽅向最⼤化 , ⽽不关⼼点之间的距离。保留了数据集的全局结构，丢弃了局部结构。

3.5. 独立分量分析（ICA）

PCA 和 ICA 之间的主要区别在于，PCA寻找不相关的因素，而 ICA 寻找独立因素。即，如果两个变量不相关，它们之间就没有线性关系。

该算法假设给定变量是一些未知潜在变量的线性混合。
它还假设这些潜在变量是相互独立的，它们不依赖于其他变量，因此它们被称为观察数据的独立分量。

3.5.1. 实施步骤

  1. 对X零均值处理
  2. 消除原始各道数据间二阶相关，以后只需要考虑高阶矩量(因为独立时各阶互累积量为0)，使很多运算过程简化。
  3. 寻求解混矩阵U，使Y=UZ，Y各道数据尽可能独立（独立判据函数G）。
  4. 由于独立判据函数G的不同，以及步骤不同，有不同的独立分量分析法。
  5. 输入球化数据z，经过正交阵U处理，输出Y=Uz
  6. 输入球化数据z，经过正交阵某一行向量处理（投影）
  7. 提取出某一独立分量
  8. 将此分量除去，按次序依次提取下去，得到所有的值
  9. 得到独立的基向量U

3.6. 局部线性嵌入（LLE）

LLE属于流形学习的一种。

流形学习是一大类基于流形的框架。我们可以认为LLE中的流形是一个不闭合的曲面。这个流形曲面有数据分布比较均匀，且比较稠密的特征（类似于水流）。

基于流形的降维算法，是将流形从高维到低维的降维过程，在降维的过程中我们希望流形在高维的一些特征可以得到保留。

3.6.1. 实施步骤

  1. 找到数据点的k个近邻点
  2. 由每个近邻点构建改样本点的局部重建矩阵
  3. 由局部重建矩阵和近邻点计算重建后的样本点

3.7. t-随机邻近嵌入（t-SNE）

PCA和ICA共有的缺陷是：只关⼼使变量的⽅向最⼤化 , ⽽不关⼼点之间的距离。保留了数据集的全局结构，丢弃了局部结构。

t 分布随机邻域嵌入（t-distributed stochastic neighbor embedding, t-SNE）实现了对非高斯分布数据降维。

谈t-SNE前，先讨论SNE。

3.7.1. SNE

SNE 使用条件概率来刻画两个数据之间的相似性，对原始数据 x1,x2,x3,…{x_1, x_2, x_3, …}x1​,x2​,x3​,…通过以下公式将数据xix_ixi​, xjx_jxj​ 间的欧几里得距离转化为概率pj∣ip_{j \mid i}pj∣i​，

pj∣i=exp⁡(−∥xi−xj∥2/2σi2)∑k≠iexp⁡(−∥xi−xk∥2/2σi2) p_{j \mid i}=\frac{\exp \left(-\left\|\mathbf{x}_{i}-\mathbf{x}_{j}\right\|^{2} / 2 \sigma_{i}^{2}\right)}{\sum_{k \neq i} \exp \left(-\left\|\mathbf{x}_{i}-\mathbf{x}_{k}\right\|^{2} / 2 \sigma_{i}^{2}\right)} pj∣i​=∑k​=i​exp(−∥xi​−xk​∥2/2σi2​)exp(−∥xi​−xj​∥2/2σi2​)​

规定 pj∣i=0p_{j \mid i}=0pj∣i​=0, 对于所有的iii， ∑jpj∣i=1\sum_{j} p_{j \mid i}=1∑j​pj∣i​=1

该算法中，当xjx_jxj​落入以xix_ixi​为中心的概率密度函数（高斯函数）时，认为xix_ixi​, xjx_jxj​ 近邻。

然而，由于 高斯核(Gaussian kernal) 使用 欧几里得距离，受维度影响很大。在高维数据集中，欧拉距离失去了分辨的能力，导致两点间pijp_{ij}pij​十分接近。因此，有时对欧几里得距离做指数化。

pij=pj∣i+pi∣j2Np_{i j}=\frac{p_{j \mid i}+p_{i \mid j}}{2 N} pij​=2Npj∣i​+pi∣j​​

pi∣jp_{i \mid j}pi∣j​（原始空间）与qi∣jq_{i \mid j}qi∣j​（嵌入空间） 越接近，代表降维算法在局部特征保留上的表现越好。用KL散度 (Kullback-Leibler divergences)建立目标函数 (cost function) 为：

C=∑iKL(Pi∣Qi)=∑i∑jpj∣ilog⁡{pj∣iqj∣i}C=\sum_{i} K L\left(P_{i} \mid Q_{i}\right)=\sum_{i} \sum_{j} p_{j \mid i} \log \left\{\frac{p_{j \mid i}}{q_{j \mid i}}\right\} C=i∑​KL(Pi​∣Qi​)=i∑​j∑​pj∣i​log{qj∣i​pj∣i​​}

由于 KL 散度的不对称性质，要使得 损失函数取到更小，原始数据中距离近的两个点的pj∣ip_{j \mid i}pj∣i​在降维后应该也较大，所以 SNE 更善于保留数据中的局部特征。
但是原始数据中距离较远的两个点如果降维后 pj∣ip_{j \mid i}pj∣i​较大，目标函数也会很小，这是 SNE 的一个缺陷，忽视了数据的全局性。

3.8. t-SNE

因此尽管 SNE 提供了很好的可视化方法，但存在不少缺陷。后续，Hinton等人提出了 t-SNE 方法。t-SNE 在嵌入空间中采用长尾的 t 分布取代高斯分布。由于它的长尾性，它更不容易受到异常点的影响而产生偏移和形变。

与SNE相同，t-SNE定义概率密度函数

pij=pj∣i+pi∣j2Np_{i j}=\frac{p_{j \mid i}+p_{i \mid j}}{2 N} pij​=2Npj∣i​+pi∣j​​

同样用 KL散度 (Kullback-Leibler divergences) 建立目标函数。

然而, t-SNE将原始空间中数据点的相似度由高斯联合分布表示，嵌入(mapping) 空间中数据点的相似度由Student t分布表示。t-SNE定义嵌入空间中数据点的相似度qijq_{ij}qij​为：

qij=(1+∥yi−yj∥2)−1∑k∑l≠k(1+∥yk−yl∥2)−1q_{i j}=\frac{\left(1+\left\|\mathbf{y}_{i}-\mathbf{y}_{j}\right\|^{2}\right)^{-1}}{\sum_{k} \sum_{l \neq k}\left(1+\left\|\mathbf{y}_{k}-\mathbf{y}_{l}\right\|^{2}\right)^{-1}} qij​=∑k​∑l​=k​(1+∥yk​−yl​∥2)−1(1+∥yi​−yj​∥2)−1​

规定 qii=0q_{ii}=0qii​=0。

通过原始空间和嵌入空间的联合概率分布的KL散度（用于评估两个分布的相似度的指标，经常用于评估机器学习模型的好坏）来评估嵌入效果的好坏，即将有关KL散度的函数作为损失函数，通过梯度下降算法最小化损失函数，最终获得收敛结果。

3.8.1. 实施步骤

  1. 对于原始数据集 X=x1,…,xnX = x_1, …, x_nX=x1​,…,xn​，先使用 PCA 算法进行预降维
  2. 根据困惑度 Perp（通常在 5-50 之间，表示一个点附近的有效近邻点个数），使用二分搜索找到一个合适的 σ\sigmaσ，作为目标函数的参数
  3. 设置迭代次数 n，学习速率 learning_rate, 为了加速优化过程和避免
  4. 陷入局部最优解，设置一个动量 α和簇间距
  5. 计算条件概率矩阵，计算 pijp_{ij}pij​ 矩阵
  6. 随机初始化 yyy
  7. 迭代 n 轮，每轮做操作如下: 计算低维度下的qijq_{ij}qij​，计算梯度

3.8.2. 缺陷

t-SNE 的不足在于它的速度较慢，且算法是随机的，它没有唯一最优解，由于 t-SNE 的目标函数是非凸的，每次得到的只是一个局部最小结果。同时 t分布只是能够保留局部特征，在结果中超出一定距离范围后的数据点相似度都很小，因此考虑 t-SNE 结果中的距离是无意义的。所以 t-SNE 难以处理其他任务，它只能作为一个降维过程的可视化工具被使用。
